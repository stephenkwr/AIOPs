Aurora Supply Co. runs promotional discount codes throughout the year — seasonal sales, welcome offers for new customers, and email subscriber perks. This article covers how codes work at checkout and how to help customers troubleshoot common issues.

### Applying a code at checkout
On the Checkout page, above the "Place Order" button, there's a field labeled "Promo code." Customers enter the code exactly as issued (codes are not case-sensitive, but must not include spaces) and select Apply. The discount recalculates immediately and appears as a line item (e.g., "Promo: WELCOME15 −$12.75") above the order total. Codes must be applied before payment is submitted — we cannot add a promo code to an order retroactively once it has been placed.

### Combining codes
Only one promo code can be applied per order. Entering a second code replaces the first rather than stacking with it. Promo codes can, however, be combined with store credit and gift card balances, since those are applied as separate payment methods rather than as discounts.

### What's excluded
Promo codes do not apply to:
- Gift card purchases
- Items already in the Clearance or Sale collection
- Aurora Outdoor Pro tents and the Heritage Oak furniture line (fixed retail pricing per manufacturer agreement)
- Shipping fees

If a cart contains both eligible and excluded items, the discount applies only to the eligible items' subtotal.

### Code not working
Most issues fall into one of four categories: the code has expired (seasonal codes typically run 14 days; welcome codes run 30 days from signup), the order doesn't meet the minimum spend (commonly $50 for welcome codes), the cart contains only excluded items, or the code has already been redeemed on that account. Agents can verify expiry and usage status in the Orders > Promotions tab of the admin console. We do not issue manual extensions or backdate discounts on expired codes; instead, direct the customer to check the Aurora Supply newsletter sign-up page for a current active offer.

### One code per customer
Welcome and referral codes (e.g., WELCOME15, 15% off, $50 minimum) are limited to one redemption per customer, tracked by account email and billing address. If a customer tries to reuse a welcome code on a new account, checkout blocks the discount with the message "This code has already been used on your account." This limit does not apply to recurring seasonal codes (e.g., SUMMER10), which any customer may use once per issue period.
